my Resume app
An application in the spirit of the MVC pattern, connected to SQL database. Allows you to create, read, update, and delete fields. It's similar to a CV generator. 
### How it works ->  


### Used tech stack:
Spring Boot,
Java 17,
Spring MVC,
Spring Data JPA,
Spring Security,
MySQL,
Bootstrap
